# React Ecommerce App

A modern ecommerce web application built using React and Vite.

## Tech Stack

- React
- Vite
- JavaScript

## Setup

npm install  
npm run dev

## Features

- Application layout
- Navigation bar
- Routing with React Router
- Home and Not Found pages
- Product listing from REST API
- Product detail view
- Add to cart functionality
- User authentication
- Protected routes
- Order pages
- Product search functionality
